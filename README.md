# Portal de Recursos Educacionais

Portal web para centralizar e organizar links de recursos educacionais gratuitos, destinado a professores, pais e alunos.

## Descrição

Projeto de extensão universitária vinculado ao Programa Institucional Educação e Tecnologia Social, alinhado ao ODS 4 - Educação de Qualidade (Meta 4.4).

## Tecnologias

- **Next.js 15** - App Router
- **TypeScript**
- **CSS Modules** + CSS Custom Properties

## Getting Started

```bash
npm install
npm run dev
```
